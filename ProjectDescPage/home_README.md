# Prediction Market Innovation - Whitepace Landing Page

A modern, responsive landing page showcasing research on prediction markets and their applications in risk management.

## Features

- **Modern Design**: Clean, professional SaaS-style layout inspired by Whitepace design
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Six Key Sections**: Comprehensive coverage of prediction market research framework
- **Interactive Navigation**: Smooth scrolling navigation with sticky header

## Structure

The webpage includes:

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **Research Framework Overview**: Introduction to the comprehensive approach
3. **Six Main Content Sections**:
   - Identification of Gaps and Use Cases
   - Mathematical & Algorithmic Framework
   - Institutional Applications
   - Prototype Operational Model
   - Oracle & Data-Extraction Protocols
   - Governance & Security Assessment
4. **Call-to-Action Section**: Encouraging engagement
5. **Footer**: Contact information and additional links

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required.

### Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

## Customization

The design uses CSS custom properties (variables) defined in `:root` for easy theming:

- `--primary-color`: Main brand color (#4F46E5)
- `--secondary-color`: Accent color (#10B981)
- `--text-primary`: Main text color
- `--text-secondary`: Secondary text color

Modify these variables in `styles.css` to change the color scheme.
